**अनन्तव्यय** is a new programming language that combines the ease of use and readability of Python with the raw power and efficiency of C. With **अनन्तव्यय**, you can write code that feels just like Python but runs with the performance of compiled languages like C. This language is designed to support all Python libraries, making it a seamless and highly efficient alternative for developers who need both simplicity and speed.

## Features 🌟

| Feature                     | Description                                                                                  |
|-----------------------------|----------------------------------------------------------------------------------------------|
| **Python-like Syntax**       | Write code just like Python, easy to read and understand.                                     |
| **C-like Performance**       | Achieves the same speed as C, making it perfect for resource-intensive tasks.                 |
| **Full Python Library Support** | Use all the Python libraries, ensuring compatibility with existing tools and frameworks.       |
| **Easy to Learn**            | If you know Python, you'll be right at home with **अनन्तव्यय**.                               |
| **Cross-Platform**           | Works across different platforms, bringing Python’s convenience with C’s performance.         |

## Usage 💻
Once released, you'll be able to execute **अनन्तव्यय** code just like Python. Here's a simple example:

### Example Code:

```
# This is an example of anantavyay code

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # Output will be 120
```
